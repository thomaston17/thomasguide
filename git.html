How to get started

  1. Create the master branch on GitHub and make sure to include the README file
  2. Copy the HTTPS clone link from GitHub
  3. Open up terminal
  4. Type "ls" command to list files
  5. Next "cd github" to change directory to where the folder is located
  6. Next "cd squarethumb" to get to the folder
  7. Next "git clone https://github.com/thomaston17/squarethumb.git"
  8. After that is done, Github will clone the folder to desktop
  9. To check other members work and sync it with your files "git pull" (to pull updated files)

Prepare files to be added

  1. "cd squarethumb"
  2. "git status" (file will display red)
  3. "git add index.html" or "git add -A" (will add all files worked on)
  4. "git status" (now file should display green if upload completed)

Commit and push changes to Github

  1. git commit -m "Added Updated Files" if you ever commit without a comment, to get out hit ESC, :wq, Enter
  2. "git push"
  3. Refresh your Github account in the browser to reflect the changes!
